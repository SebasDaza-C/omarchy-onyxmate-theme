# OnyxMate
A sophisticated monochromatic theme for modern environments.
Pure black and white, with perfect shades of gray.
Designed for clarity, focus, and elegance.
Minimalist approach that enhances productivity while maintaining visual harmony.

## Features
- Pure monochromatic color scheme
- Elegant and modern design
- Focus on readability and visual comfort
- Minimalist interface elements
- Carefully crafted gray scales using OKLCH color space
- Perfect for both light and dark environments



# Installation Theme

To install this theme, use the omarchy-theme-install command:

```bash
omarchy-theme-install https://github.com/SebasDaza-C/omarchy-onyxmate-theme.git
```

## Vesktop Theme
For Vesktop users, install the theme by copying it to your Vesktop themes directory:
```bash
cp system24-onyxmate.theme.css ~/.config/vesktop/themes/system24-onyxmate.theme.css
```
